## Core Style

OpenClaw images are sparse 16:9 horizontal article illustrations where the red logo-derived OpenClaw mascot performs one cognitive action inside an original article metaphor.

## Must Have

- 16:9 horizontal article illustration by default.
- Clean white or near-white background with generous whitespace.
- One core idea per image.
- OpenClaw performs the core conceptual action.
- Loose black linework with visible hand-drawn looseness.
- Restrained red accents tied to the uploaded logo body.
- Restrained cyan accents tied to the pupils or one small action emphasis.
- Concise visible labels, usually 2-6 total, copied in the user's requested language.
- Sparse supporting objects selected for the article idea.
- OpenClaw recognition stays clear inside the simplified drawing style.

## Uploaded-Logo Identity Markers

Preserve these uploaded OpenClaw logo markers together:

- red round body
- side claw-like arms
- two antennae
- black eyes
- cyan pupils
- short legs

Uploaded-logo identity matters more than rendering detail density. Keep OpenClaw simple enough for article explanation while preserving the red body, side claw-like arms, antennae, eyes, cyan pupils, and short legs.

## Composition

- Use a single visual structure: Workflow, System Local View, Before/After, Character State, Concept Metaphor, Method Layers, Map Route, or Mini Comic.
- Make the OpenClaw action physically readable: inspect, grip, sort, bridge, tune, stamp, carry, map, connect, shield, compare, mark, untangle, weigh, guide, assemble, or repair.
- Use article metaphor props sparingly: maps, stepping stones, bridges, hooks, claws, levers, knots, shelves, signposts, lamps, shields, stamps, keys, trails, gates, scales, buckets, envelopes, threads, lenses, compasses, or small hand-built machines.
- Treat supporting objects as article metaphors for judgment, review, handoff, hidden coupling, failure mode, decision gates, framing, sequencing, or conceptual repair.
- Leave empty space around OpenClaw's action so the metaphor reads at a glance.
- Keep labels short and separated from OpenClaw's face, antennae, claws, pupils, and legs.
- OpenClaw cognitive-action participation gate: OpenClaw performs the core cognitive action, so the image loses its meaning if OpenClaw is removed.
