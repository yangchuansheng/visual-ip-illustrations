# Xiaohei IP

## Character Definition

Xiaohei is the fixed visual IP for Ian-style article-body illustrations.

Xiaohei appears by default in each Xiaohei image. Xiaohei is a serious absurd worker participating in the system's operation.

Legacy compatibility marker: `小黑承担核心动作`.

## Appearance

- Small solid-black absurd creature.
- White dot eyes.
- Thin legs, sometimes thin arms.
- The body can read as a cylinder, black bean, black box, funnel, shadow, hole, or black block inside a machine.
- Slightly irregular hand-drawn contour.
- Blank, deadpan, calm, serious expression.

## Personality

- Serious while doing something slightly absurd.
- Feels like a quiet system operator.
- Dry humor.
- Slightly clumsy, still competent.
- Looks genuinely responsible for one job inside the sketch.

## Common Responsibilities

Give Xiaohei the core action:

- Carry material.
- Pull lines that gather information sources.
- Get stuck in a breakpoint.
- Operate a "judgment" lever inside a machine.
- Become a filtering funnel.
- Cut open a "material fish".
- Stamp handoff copy.
- Hold a handoff path.
- Hold a warning sign beside pitfalls.
- Reach out from a hole and fail to catch content.
- Move bricks, build a bridge, open a door, sort, or record.

## Vetoes

- Overly cute mascot treatment.
- Children's cartoon treatment.
- Complex clothing, meme expressions, or shiny eyes.
- Xiaohei standing in a corner as a spectator.
- Xiaohei overpowering the structural explanation.
- Commercial, rounded, polished Xiaohei rendering.

Legacy QA marker: `小黑像吉祥物、表情包或可爱卡通`.

## Participation Test

If removing Xiaohei leaves the core metaphor fully intact, Xiaohei is too decorative. Rewrite the prompt so Xiaohei becomes the action subject.
