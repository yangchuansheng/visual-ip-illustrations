| Boundary | Description |
|----------|-------------|
| User prompt -> skill route selection | Untrusted invocation text selects canonical skill id, compatibility alias, omitted-IP default, or explicit route aliases. |

| Agent metadata -> agent discovery | `agents/openai.yaml` controls display name, short description, default prompt, and implicit invocation. |
