# Quick Task 260617-l3a: Move Localized README Files Into A Readmes Directory

## Goal

Keep the English `README.md` at the repository root, move every localized README into a dedicated `readmes/` directory, and preserve all cross-README and local asset links.

## Tasks

1. Move localized README files
   - files: `README.*.md`, `readmes/`
   - action: Move all non-English README variants into `readmes/`.
   - verify: `find . -maxdepth 2 -name 'README*.md' -print | sort` shows only `README.md` at the root and localized README files under `readmes/`.

2. Rewrite README links
   - files: `README.md`, `readmes/README.*.md`
   - action: Update the root language selector to point into `readmes/`, update localized selectors to point back to `../README.md`, and adjust local links/images from moved files.
   - verify: Run the local Markdown link checker and confirm every README link resolves inside the repository.

3. Update validation coverage
   - files: `scripts/validate-skill-package.mjs`, `scripts/validate-skill-package.test.mjs`
   - action: Make README variant discovery and Chinese gallery validation aware of the nested localized README layout.
   - verify: `node scripts/validate-skill-package.mjs`, `node --test scripts/validate-skill-package.test.mjs`, and `git diff --check`.
