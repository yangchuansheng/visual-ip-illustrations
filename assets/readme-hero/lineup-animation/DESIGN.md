# Visual IP Lineup Animation Design

## Style Prompt

Animate the existing Visual IP Illustrations README hero image as a clean hand-drawn character lineup. Preserve the white paper canvas, rough black linework, soft pencil/marker color, generous whitespace, and friendly route-local character identities. Motion should feel like subtle character acting inside a README hero banner, with no new copy, badges, logos, gradients, or poster framing.

## Colors

- `#ffffff` — paper canvas
- `#111111` — hand-drawn linework
- `#d9472f` — OpenClaw and small warm accents
- `#1f5f9d` — Seal hoodie/navy accents
- `#4eb3d3` — Go Gopher blue
- `#d66f2e` — Ferris orange

## Typography

- Keep the canvas text-free.
- Metadata and source comments use English only.

## Motion

- Short seamless loop, 10 seconds.
- Character movements stay subtle: bob, wave, lean, bounce, point, and prop wiggle.
- Use transform-only GSAP animation.
- Keep the original lineup readable at every frame.

## Guardrails

- Keep the frame title-free, label-free, caption-free, UI-free, chart-free, and logo-free.
- Keep every face fully visible and every character inside the canvas.
- Use the original white hand-drawn visual language across the whole animation.
- Keep route identities clean: Go Gopher stays mascot-only, Sealos Seal stays logo-free, and Tom appears as a solo protected-character route.
